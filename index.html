Bring your laptop to class

Macbooks are the preferred laptop for the class; however, Windows machines will work just fine. Chromebooks are not recommended. You do not need a new or top of the line computer to learn web development. Minimum screen size of 13 inches recommended.


Download a text-editor

Brackets.io is a new, free text editor available for Mac or PC. We will be using Brackets in class; however, feel free to try other editors including Atom, Sublime Text 3, Notepad, the list goes on and on... And don't worry about using the editor yet, we will cover that in the first class.


Sign up for codeacademy.com

Codeacademy is free. Try the first section of the HTML&CSS course called "HTML Basics, Fundamentals of HTML". It is a good hands-on overview if you have never tried HTML before. Online tutorials can become very frustrating when you have questions, so take it slow and don't try to do to much at once.



Fall in love with a book...

In order to become a web developer, you really need to fall in love with coding. If you don't already have an intro to HTML book, you might want to sample some on your Kindle (or visit a bookstore) until you find the perfect book. It is very important that you enjoy the author's writing style, so please search carefully. Most introduction to HTML books have the necessary information, and Hackingtons will always provide free web resources for every lesson. Some HTML books are not available on the Kindle because of graphic limitations. If I were to 'choose' a class textbook, it would be HTML&CSS Design and build websites by Jon Duckett". This book is not available on Kindle.
